# Number Row Characters
* ~ - Tilde
* ! - Exclamation point
* @ - At symbol
* \# - Hash tag
* $ - Dollar sign (currency)
* % - Percent and modulo symbol
* ^ - Caret symbol
* & - Ampersand symbol
* \* - Asterisk symbol
* ( - Open parenthesis
* ) - Close parenthesis
* _ - Underscore
* \+ - Plus sign
